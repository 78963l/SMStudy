C/C++ 처럼 메모리를 직접 할당/해제할 필요가 없음.  
python에는 Gabage Collection이 있어 자동으로 수집하고 해제함.(java와 동일)

하여 메모리에 문제가 생길 수 있어 None타입으로 변경 후 del함수를 사용하는 것이 바람직함.
